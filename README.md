# 🔮 SIEM-SIGHT

A browser-based Security Log Analyzer & Threat Detection Platform that demonstrates SOC analyst workflows, MITRE ATT&CK mapping, and Sigma rule generation.

## 🚀 Live Demo

https://adamanuhu-star.github.io/siem-sight

## 📊 Features

- **Multi-Source Log Ingestion** — Auth, firewall, and web server logs
- **Real-Time Threat Detection** — Brute-force, port scans, SQL injection
- **MITRE ATT&CK Mapping** — Maps events to real attack techniques
- **Attack Timeline** — Chronological visualization of critical events
- **Attacker Profiling** — Identifies top threat actors by volume
- **Sigma Rule Generator** — Auto-creates detection rules from patterns

## 🛠️ Tech Stack

- HTML5, Tailwind CSS, Vanilla JavaScript
- No external dependencies
- Simulated log data for educational purposes

## 🎯 SOC Concepts Demonstrated

- **Log Correlation:** Identifying patterns across multiple data sources
- **Brute Force Detection:** Threshold-based anomaly detection
- **MITRE ATT&CK:** Mapping observed events to known attack techniques
- **Sigma Rules:** YAML-based detection rule syntax
- **Incident Response:** Alert triage and recommendation workflows
